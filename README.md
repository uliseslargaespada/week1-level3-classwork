# week1-level3-classwork

This repository has the content that we've been working on Week 1 of Level 3

## Installation

1. Clone the repository
```bash
git clone repo_url
```

2. Install the project dependencies
```bash
npm install
```

3. Run the dev server
```bash
npm run dev
```

## Utilities
1. Testing
```bash
npm run test
```

2. Check environment
```bash
npm run check:node
```
